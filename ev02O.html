Crea una sola gráfica donde se comparen el número de bases de ADN que componen todas las variantes del virus.

cont <- function(seq){
  cont1A <-0
  cont1T <-0
  cont1C <-0
  cont1G <-0
  
  nl <- length(seq)
  
  for(i in seq){
    
    if(i=="A" || i=="a"){
      cont1A = cont1A + 1
    }
    else if(i=="T" || i=="t") {
      cont1T = cont1T + 1
    }
    else if(i=="C" || i=="c") {
      cont1C = cont1C + 1
    }
    else if(i=="G" || i=="g") {
      cont1G = cont1G + 1
    }
  }
  
  seq.x1<- c(cont1A,cont1T,cont1G, cont1C)
  return(seq.x1)
}

Interpretacion

Como se puede observar en la grafica, la hebra del pato cuenta con una mayor cantidad de adenina citosina guanina y timina en comparacion con los otros animales, esto se debe a que cuenta con un ARN menos homologo que con las diferentes especies de murcielagos y animales de casa. Por otro lado, en el arbol filogenetico la especie de murcielagpo “Rousettus” no tiene tanta similitud con sus otras variantes, debido a la zona geografica en la que suelen encontrarse y las situaciones a las que se exponen y por lo mismo representa una mayor alteracion en su codigo genetico.
